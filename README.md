# NetworkScanner

### Description
Simple commandline networkscanner.
 
 
Only works with 192.168.xxx.xxx-ip-addresses.



#### Technologies
java and maven.


#### To run

1. execute the "runscript"(bash-script)

    or

2. run the commands:

```sh
$ mvn clean package
$ java -jar .../yourFriend/target/yourFrend-1.0-SNAPSHOT.jar
```
